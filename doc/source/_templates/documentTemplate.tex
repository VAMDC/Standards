\textbf{Abstract:} & 
This document describes the layout of VAMDC modified version of
International Atomic Energy Agency's XML Schema for Atomic, Molecular and Solid Data (XSAMS).
The modified schema is the working data model for the VAMDC consortium.  
The adopted name for the modified schema is XSAMS-VAMDC.
The internal version of the schema described is 0.2.

The document is based on the official IAEA XSAMS documentation.
